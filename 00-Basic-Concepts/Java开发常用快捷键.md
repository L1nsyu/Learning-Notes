# ⌨️ IDEA 高效快捷键速查表 (Java 开发者必备)

> **核心心法**：忘掉鼠标，让双手留在键盘上。

---

## 一、 核心重构与代码生成 (写代码最快的方式)

IDEA 最强大的地方在于它能帮你写代码和改代码。

- **`Alt + Enter` (万能键)**：智能提示/自动修复。报错了按它，导包按它，警告按它，它是 IDEA 里最重要的快捷键。
- **`Alt + Insert` (生成键)**：自动生成构造方法、`getter/setter`、`toString()`、重写方法等。
- **`Shift + F6` (重命名)**：安全地批量重命名变量、方法或类名。它会自动帮你修改所有引用了该名字的地方，绝不会漏掉。
- **`Ctrl + Alt + M` (抽取方法)**：将一段复杂的代码片段提取成一个独立的方法，保持主方法整洁。
- **`Ctrl + Alt + V` (提取变量)**：快速为右边的表达式生成左边的变量声明。（例如写完 `new User()` 按此键，会自动生成 `User user = new User();`）
- **`Ctrl + Alt + C` (提取常量)**：将魔法值（如固定的字符串或数字）提取为全局常量。

---

## 二、 批量修改与光标操作 (极客专属)

- **`Ctrl + Alt + Shift + J` (选中所有相同词)**：选中某个单词后按此键，可以瞬间选中当前文件内所有相同的单词，实现多光标同步批量修改。
- **`Alt + J` (逐个选中相同词)**：选中单词后，按一次 `Alt + J` 选中下一个相同的词，可以精准控制要修改哪几个。
- **`Ctrl + D` (向下复制)**：直接将当前行（或选中的代码块）原封不动地向下复制一份。
- **`Ctrl + Y` (删除当前行)**：直接删掉光标所在的整行代码，不需要选中。
- **`Shift + Enter` (向下换行)**：无论光标在当前行的哪个位置（即使在中间），按此键都会直接在下方新建一个空白行，并将光标移动过去，非常流畅。
- **`Alt + Shift + ↑ / ↓` (上下移动代码)**：将当前行（或选中的代码块）向上或向下移动。

---

## 三、 快速导航与查找 (找代码不用鼠标)

- **`双击 Shift` (Search Everywhere)**：全站搜索。无论是类名、文件名、还是 IDEA 的设置项，输入名字就能搜到。
- **`Ctrl + N` (查找类)**：专门用来搜索 Java 的 Class 文件。
- **`Ctrl + B` (跳转到定义)**：光标放在方法或变量上按此键（等同于 `Ctrl + 鼠标左键`），可以直接跳到它被定义的地方。
- **`Ctrl + Alt + B` (跳转到实现)**：🌟 **极其重要**！在 Java 三层架构中，如果光标在 Service **接口**的方法上按 `Ctrl + B`，只会跳到接口定义；但按 `Ctrl + Alt + B`，就能直接跳到 `ServiceImpl` **实现类**里的具体代码。
- **`Alt + F7` (查找使用情况)**：查找当前类、方法或变量在整个项目中被哪些地方调用了 (Find Usages)。
- **`Ctrl + E` (最近文件)**：弹出最近打开过的文件列表，方便在两个文件之间快速来回切换。

---

## 四、 格式化与基础环境

- **`Ctrl + Alt + L` (格式化代码)**：一键让杂乱无章的代码变得缩进对齐、整整齐齐。**（每次提交代码前必按）**
- **`Ctrl + /` (单行注释)**：注释或取消注释当前行 (`//`)。
- **`Ctrl + Shift + /` (多行注释)**：将选中的代码块用 `/* ... */` 注释起来。
- **`Ctrl + . (句号)`**：系统输入法快捷键，在中文输入状态下，快速切换**中英文标点符号**（经常为了敲一个英文分号 `;` 而切换输入法，用这个快捷键可以只切标点不切语言）。
