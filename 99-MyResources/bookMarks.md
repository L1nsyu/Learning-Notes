## 🛠️ 在线开发工具 (Dev Tools)
- **[Apifox](https://apifox.com/)** - 接口调试、文档生成，比 Postman 好用。
- **[Json.cn](https://www.json.cn/)** - JSON 格式化与校验，看复杂数据必备。
- **[Cron 生成器](https://cron.qqe2.com/)** - 定时任务表达式生成，防写错。
- **[Maven 仓库](https://mvnrepository.com/)** - 找 Jar 包依赖坐标 (GAV) 必去。
- **[时间戳转换](https://tool.lu/timestamp/)** - 秒/毫秒转换，查 Bug 用。
- **[命名神器『ValGod』](https://valgod.vercel.app/)** - 程序员翻译工具，解决变量命名困难。

## 📚 官方文档 & 手册 (Documentation)
- **[Spring Boot 官方文档](https://spring.io/projects/spring-boot)** - 遇到报错查源头。
- **[MDN Web Docs](https://developer.mozilla.org/zh-CN/)** - 前端 (HTML/CSS/JS) 的权威圣经。
- **[Java 全栈知识体系](https://pdai.tech/)** - 也就是 PDAI，非常系统的 Java 路线。

## 🎨 设计与素材 (Design & Assets)
- **[IconFont](https://www.iconfont.cn/)** - 阿里巴巴矢量图标库，找图标方便。
- **[Unsplash](https://unsplash.com/)** - 免费无版权的高清图片，做 PPT 或 Demo 用。

## ⚡ 学习与社区 (Community)
- **[Bilibili (黑马程序员)](https://space.bilibili.com/37974444)** - 视频教程根据地。
- **[GitHub Trending](https://github.com/trending)** - 看看最近大家都在写什么开源项目。

## 👍 方便且常用的小工具 (Tools)
- **[I Love PDF](https://www.ilovepdf.com/zh-cn)** - PDF 爱好者的在线工具，支持合并、拆分、转换格式等。
- **[哔哩哔哩视频下载](https://snapany.com/zh/bilibili)** - 方便地下载 B 站视频。
- **[力扣 (LeetCode)](https://leetcode.cn/)** - 程序员刷算法题、准备面试的必备网站。
- **[樱花猫 (Sakura Cat)](https://sakura-cat1.com/)** - 樱花猫梯子。
- **[LMSYS Chatbot Arena](https://lmarena.ai/leaderboard)** - 查看各大语言模型（如 GPT-4, Llama）的实时对战排行榜。
- **[去码头](https://www.doiduoyi.com/)** - 一个集聊天、摸鱼、游戏、学技术于一体的社区网站。
